## Installation

The following command will add the Drawer component into your `components/ui` folder

```bash
$ npx flicker-ui@latest add drawer
```

<br />

## Usage

Use the `direction` prop to specify which direction should the drawer open from.

```js /items="center"/ /justify="center"/
<Badge variant="success" size="md">Badge</Badge>
```
