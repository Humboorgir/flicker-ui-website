## Installation

The following command will add the Tooltip component into your `components/ui` folder

```bash
$ npx flicker-ui@latest add tooltip
```

<br />

## Usage

```js /TooltipTrigger/ /TooltipContent/ /Tooltip/
<Tooltip>
  <TooltipTrigger> tooltip trigger text here </TooltipTrigger>
  <TooltipContent> tooltip content here</TooltipContent>
</Tooltip>
```
