## Installation

The following command will add the Badge component into your `components/ui` folder

```bash
$ npx flicker-ui@latest add badge
```

<br />

## Usage

Use the `variant` prop to specify badge variant, and `size` to specify badge size.

```js /items="center"/ /justify="center"/
<Badge variant="success" size="md">Badge</Badge>
```
