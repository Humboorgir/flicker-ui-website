## Installation

The following command will add the Column component into your `components/ui` folder

```bash
$ npx flicker-ui@latest add column
```

<br />

## Usage

You can specify `align-items` with `items`, and `justify-content` with `justify`

```js /items="center"/ /justify="center"/
<Column items="center" justify="center">
  <div className="bg-red-500 h-16 w-16 rounded mr-3"></div>
  <div className="bg-red-500 h-16 w-16 rounded mr-3"></div>
  <div className="bg-red-500 h-16 w-16 rounded"></div>
</Column>
```
