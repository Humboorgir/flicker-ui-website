## Installation

The following command will add the Input component into your `components/ui` folder

```bash
$ npx flicker-ui@latest add input
```

<br />

## Usage

Used just like the input element. You can specify properties like `name`, `id`, `className` etc.
width is set to 200px by default
```js /placeholder="Email address"/ /name="email"/ /id="email"/
<Input placeholder="Email address" name="email" id="email"/>
```
