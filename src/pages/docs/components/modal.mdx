## Installation

The following command will add the Modal component into your `components/ui` folder

```bash
$ npx flicker-ui@latest add modal
```

<br />

## Usage

The modal component takes two properties: `open` and `handleClose`, `open`
determines whether the modal is open or not, and `handleClose` is a function that 
sets `open` to false
```js /items="center"/ /justify="center"/
<Modal>
</Modal>
```
