## No installation needed
FlickerUI is not a component library, it's a collection of premade tailwindcss components. <br/>
You can add each component to your project by using the FlickerUI CLI:
```bash "[component]"
npx flicker-ui@latest add [component]
```

<br/>
<br/>

## CLI Usage
View a list of all the available commands
```bash
npx flicker-ui@latest --help
```
<br/>

View a list of all the available components
```bash
npx flicker-ui@latest list
```
<br/>

Add a component to your project
```bash 
npx flicker-ui@latest add [component]
```
