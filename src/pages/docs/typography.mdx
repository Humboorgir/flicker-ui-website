## Installation

The following command will add the Typography component into your `components/ui` folder

```bash
$ npx flicker-ui@latest add typography
```

<br />

## Usage

It comes with 6 different variants: h1, h2, h3, h4, h5, p and lead
specify variants with the `variant` property
```js /variant="p"/
<Typography variant="p">Hello there my friend!</Typography>
```
