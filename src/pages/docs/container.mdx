## Installation

The following command will add the Container component into your `components/ui` folder

```bash
$ npx flicker-ui@latest add container
```

<br />

## Usage

The container component is treated as a div element,
so you can add customizations like className to it

```js /className="bg-red-500"/
<Container className="bg-red-500">
 {/** Page content */}
</Container>
```
